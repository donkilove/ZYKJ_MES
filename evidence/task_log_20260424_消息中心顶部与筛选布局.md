- 调研摘要：`MessageCenterHeader` 原先将全部按钮放在同一 `MesToolbar` 尾部；`MessageCenterFilterSection` 仅透传 `child`；`MessageCenterOverviewSection` 按可用宽度等分四卡，缺少紧凑约束。
- 执行摘要：
  - 新增 `message_center_action_bar.dart`，把顶部按钮拆成独立主操作区并按动作分组。
  - `message_center_header.dart` 改为“页头信息 -> 错误提示 -> 主操作区”分层结构。
  - `message_center_filter_section.dart` 在不改主页面逻辑的前提下，重组现有 `Wrap` 为常用筛选与辅助筛选两层，并增加 `message-center-secondary-filters` key。
  - `message_center_overview_section.dart` 保留 4 个 `MesMetricCard`，通过列数与卡片宽度上限压缩布局密度。
  - `message_center_page_test.dart` 先加失败测试，再验证新 key 与紧凑概览要求。
- 验证摘要：
  - 红灯：`flutter test test/widgets/message_center_page_test.dart -r expanded` 首轮失败，原因是缺少 `message-center-primary-actions`。
  - 绿灯：格式化后再次运行同一命令，16 项测试全部通过。
