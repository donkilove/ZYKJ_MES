## 6. 实施结果

- 已完成改动：
  1. 新增 `sys_user_export_task` 模型与 Alembic 迁移，支持异步导出任务持久化。
  2. 新增后端导出任务服务，使用 `BackgroundTasks` 生成文件并维护 `pending/processing/succeeded/failed/expired` 状态。
  3. 用户导出新增任务接口、任务列表接口、任务详情接口与下载接口。
  4. 导出文件写入 `backend/runtime_exports/user/`，文件名带数据范围、角色码与时间戳。
  5. 前端用户管理页改为“创建导出任务 -> 查看导出任务 -> 下载文件”，并提供任务轮询与状态展示。
  6. 通用导出文件服务扩展为支持 CSV/Excel 二进制保存。
