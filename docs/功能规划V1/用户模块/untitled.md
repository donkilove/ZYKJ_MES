# 用户模块需求说明

## 1. 文档信息

- 模块名称：用户模块
- 文档状态：最终可评审稿
- 版本：V1.0
- 目标：完成用户、注册审批、角色、权限、个人中心、登录日志/在线会话及审计相关基础能力建设，满足系统后台用户管理与权限控制需求。

---

## 2. 模块范围

本模块本期包含以下页面：

1. 用户管理页面
2. 注册审批页面
3. 角色管理页面
4. 功能权限配置页面
5. 操作审计日志页面
6. 个人中心/账号设置页面
7. 登录日志/在线会话页面

> 说明：
> 
> - 以上 1～7 页面均纳入本期上线范围。
> - 工段管理页面不属于用户模块，归属工艺模块。
> - 用户模块内涉及“工段”的地方仅作为引用字段使用，工段数据实时来源于工艺模块。

---

## 4. 角色定义

### 4.1 系统内置角色

系统默认内置以下角色：

- 系统管理员
- 生产管理员
- 品质管理员
- 操作员
- 维修员

### 4.2 角色规则

- 系统内置角色默认存在。
- 系统内置角色不允许删除。
- 系统内置角色不允许修改名称。
- 系统内置角色允许配置权限。
- 支持新增自定义角色。
- 自定义角色允许编辑、停用、删除；已绑定用户的自定义角色不允许删除。
- 用户与角色为单选关系，一个用户仅能绑定一个角色。

## 5. 用户管理页面

### 5.1 页面目标

用于统一管理系统内全部用户，支持搜索、查看、新建、编辑、停用、启用、删除、重置密码、导出等操作。

### 5.2 页面功能

#### 5.2.1 搜索/筛选功能

支持以下筛选条件：

- 用户名搜索
- 角色筛选
- 工段筛选
- 账号状态筛选

#### 5.2.2 顶部操作区

支持以下按钮：

- 新建用户
- 进入角色管理页面
- 导出用户列表（CSV/Excel）

> 本期不支持用户导入。

### 5.3 新建用户

#### 5.3.1 表单字段

| 字段   | 类型   | 必填  | 说明                   |
| ---- | ---- | --- | -------------------- |
| 用户名  | 输入框  | 是   | 全局唯一                 |
| 用户密码 | 密码框  | 是   | 新建时填写                |
| 用户角色 | 单选下拉 | 是   | 单选                   |
| 用户工段 | 单选下拉 | 否   | 仅操作员角色可选；数据实时来源于工艺模块 |
| 账号状态 | 单选   | 否   | 默认启用                 |
| 备注   | 输入框  | 否   | 便于管理                 |

#### 5.3.2 业务规则

- 用户名必须唯一。
- 用户角色为“操作员”时，用户工段必填。
- 用户角色为非“操作员”时，用户工段字段不可编辑，展示为 `/`。
- 用户工段下拉数据实时来源于工艺模块，仅展示启用状态的工段。
- 新建成功后，用户进入用户列表，默认按数据库 `id` 升序展示。

### 5.4 用户列表展示

#### 5.4.1 列表要求

- 列表需带表头。
- 列表区域支持滚动。
- 按数据库 `id` 字段排序展示，默认升序。
- 支持空数据占位展示。

#### 5.4.2 表头字段

| 表头   | 说明             |
| ---- | -------------- |
| 用户名  | 展示用户名          |
| 角色   | 展示当前用户角色       |
| 工段   | 未分配或不适用时展示 `/` |
| 在线状态 | 在线显示绿色；离线显示灰色  |
| 账号状态 | 展示启用/停用        |
| 操作   | 下拉操作菜单         |

#### 5.4.3 操作菜单

下拉菜单包含：

- 编辑
- 停用
- 启用
- 重置密码
- 删除

### 5.5 编辑用户

#### 5.5.1 可编辑字段

- 用户名（仅系统管理员可修改）
- 用户角色
- 用户工段
- 账号状态
- 备注

#### 5.5.2 编辑规则

- 仅系统管理员可修改用户名，普通用户（非系统管理员）不得修改用户名。
- 用户密码不通过“编辑用户”直接修改，管理员修改密码统一使用“重置密码”功能。
- 若角色修改为“操作员”，则需选择工段。
- 若角色由“操作员”改为其他角色，则原工段自动清空。
- 用户工段数据来源于工艺模块，不在本模块内维护。

### 5.6 重置密码

- 管理员可对指定用户执行重置密码操作。
- 重置密码需二次确认。
- 重置后的密码需符合系统密码规则。
- 重置密码操作需记录审计日志。

### 5.7 停用、启用与删除规则

#### 5.7.1 停用

- 停用后用户不可登录系统。
- 停用后保留用户历史数据。
- 已停用用户可重新启用。

#### 5.7.2 启用

- 启用后用户可重新登录系统。
- 启用操作需记录审计日志。

#### 5.7.3 删除

- 本期用户删除统一采用逻辑删除，不提供物理删除能力。
- 删除后用户不可登录系统。
- 已删除用户默认不在用户列表中展示。
- 删除前需二次确认。
- 删除操作需记录审计日志。

### 5.8 导出规则

- 本期支持用户列表导出。
- 导出格式支持 CSV、Excel。
- 导出范围为当前筛选结果。

## 6. 注册审批页面

### 6.1 页面目标

用于处理待审批注册申请，支持审批通过、驳回。

### 6.2 页面功能

#### 6.2.1 列表要求

- 展示全部待审批注册用户。
- 列表需带表头。
- 列表区域支持滚动。
- 按数据库 `id` 字段排序展示，默认升序。

#### 6.2.2 表头字段

| 表头   | 说明          |
| ---- | ----------- |
| 用户名  | 注册申请用户名     |
| 密码   | 明文展示        |
| 申请状态 | 待审批/已通过/已驳回 |
| 操作   | 下拉操作菜单      |

### 6.3 操作菜单

下拉菜单包含：

- 通过
- 驳回

### 6.4 通过操作

#### 6.4.1 通过表单字段

| 字段   | 类型   | 必填  | 说明                   |
| ---- | ---- | --- | -------------------- |
| 用户名  | 输入框  | 是   | 默认带出申请用户名            |
| 用户密码 | 密码框  | 是   | 默认带出申请密码             |
| 用户角色 | 单选下拉 | 是   | 单选                   |
| 用户工段 | 单选下拉 | 否   | 仅操作员角色可选；数据实时来源于工艺模块 |

#### 6.4.2 通过规则

- 审批通过时需校验用户名唯一性。
- 用户角色为“操作员”时，需选择工段。
- 用户工段下拉数据实时来源于工艺模块，仅展示启用状态的工段。
- 审批通过后，注册申请状态更新为“已通过”，并生成正式用户账号。

### 6.5 驳回操作

#### 6.5.1 交互要求

- 点击“驳回”后弹出确认弹窗。
- 驳回时支持填写驳回原因。

#### 6.5.2 驳回规则

- 驳回后申请状态更新为“已驳回”。
- 保留驳回记录，不直接删除申请数据。

### 6.6 审计要求

- 审批通过操作需写入审计日志。
- 审批驳回操作需写入审计日志。

## 7. 角色管理页面

### 7.1 页面目标

用于维护角色信息，管理角色的启用状态、说明及基础属性。

### 7.2 页面功能

#### 7.2.1 新建角色

支持以下字段：

| 字段   | 类型  | 必填  | 说明         |
| ---- | --- | --- | ---------- |
| 角色名称 | 输入框 | 是   | 角色名称唯一     |
| 角色说明 | 输入框 | 否   | 描述角色用途     |
| 角色类型 | 单选  | 是   | 系统内置 / 自定义 |
| 状态   | 单选  | 否   | 默认启用       |

> 说明：新建角色时实际仅允许创建“自定义”角色，系统内置角色由系统预置。

#### 7.2.2 角色列表

| 表头    | 说明        |
| ----- | --------- |
| 角色名称  | 角色名称      |
| 角色说明  | 角色说明      |
| 角色类型  | 系统内置/自定义  |
| 关联用户数 | 当前角色下用户数量 |
| 状态    | 启用/停用     |
| 操作    | 编辑、停用、删除  |

### 7.3 角色规则

- 系统内置角色不可删除。
- 系统内置角色不可修改名称。
- 系统内置角色可配置权限。
- 自定义角色支持新增。
- 已绑定用户的角色不允许删除。

## 8. 功能权限配置页面

### 8.1 页面目标

用于按角色配置各业务模块的功能权限。

### 8.2 页面范围

#### 8.2.1 模块选择下拉栏

支持以下模块：

- 用户模块
- 产品模块
- 工艺模块
- 生产模块
- 品质模块
- 设备模块

#### 8.2.2 角色选择下拉栏

- 支持选择需配置权限的角色。

### 8.3 页面功能

#### 8.3.1 能力包展示功能

根据所选模块与角色，展示对应能力包权限项，支持勾选保存。

#### 8.3.2 本页面不包含的功能

当前版本明确去掉以下功能：

- 搜索
- 当前版本显示
- 生效预览
- 查看变更历史
- 查看最近保存结果
- 恢复上次保存前草稿
- 套用岗位模板

### 8.4 权限粒度

本期权限粒度包含：

- 页面查看
- 新建
- 编辑
- 删除
- 审批
- 配置

> 本期不包含数据范围权限（仅本人/仅本工段/全部）。

### 8.5 特殊规则

#### 8.5.1 系统管理员规则

- 系统管理员不再固定全权限。
- 系统管理员固定拥有“功能权限配置页面”访问权限。
- “功能权限配置页面”权限仅系统管理员角色可拥有、可编辑。
- “功能权限配置页面”权限不出现在能力包中。
- 非系统管理员角色不可拥有该权限。

#### 8.5.2 保存规则

- 修改权限后需点击“保存”生效。
- 保存成功后写入审计日志。
- 系统内必须至少保留一个可进入功能权限配置页面的系统管理员账号。

- 新建用户
- 编辑用户
- 停用用户
- 启用用户
- 删除用户
- 重置密码
- 审批通过
- 审批驳回
- 新建角色
- 编辑角色
- 删除角色
- 权限配置保存
- 用户修改本人密码

### 9.3 列表字段

| 表头   | 说明              |
| ---- | --------------- |
| 操作时间 | 操作发生时间          |
| 操作人  | 执行操作的用户         |

## 10. 个人中心/账号设置页面

### 10.1 页面目标

用于当前登录用户查看本人账号信息、维护个人账号安全，并查看本人当前登录概况。

本页面仅提供“本人信息查看”和“本人密码修改”能力，不提供角色、工段、权限、账号状态等组织属性的自助修改能力。

### 10.2 页面组成

页面由以下 3 个区域组成：

1. 基本信息区
2. 安全设置区
3. 会话概览区

### 10.3 页面功能

#### 10.3.1 基本信息查看

支持查看以下本人信息：

- 当前登录账号用户名
- 当前登录账号角色
- 当前登录账号所属工段
- 当前登录账号状态
- 账号创建时间
- 最近一次登录时间
- 最近一次登录 IP
- 密码最近修改时间

> 说明：
> 
> - 当前工段仅展示用户已绑定工段信息，工段数据来源于工艺模块。
> - 非操作员角色未绑定工段时，展示为 `/`。

#### 10.3.2 本人密码修改

支持当前登录用户修改本人密码，包含以下能力：

- 输入原密码
- 输入新密码
- 输入确认新密码
- 提交修改
- 修改成功后强制重新登录
- 修改失败时提示失败原因

#### 10.3.3 当前会话概览

支持查看当前登录会话的概要信息：

- 当前登录时间
- 最近活跃时间
- 当前会话状态
- 当前会话剩余有效时长
- 退出当前登录

### 10.4 页面字段

| 字段        | 所属区域  | 是否可编辑 | 说明             |
| --------- | ----- | ----- | -------------- |
| 用户名       | 基本信息区 | 否     | 仅展示当前登录用户用户名   |
| 当前角色      | 基本信息区 | 否     | 仅展示            |
| 当前工段      | 基本信息区 | 否     | 仅展示，未绑定时显示 `/` |
| 账号状态      | 基本信息区 | 否     | 仅展示，如启用/停用     |
| 账号创建时间    | 基本信息区 | 否     | 仅展示            |
| 最近一次登录时间  | 基本信息区 | 否     | 仅展示            |
| 最近一次登录 IP | 基本信息区 | 否     | 仅展示            |
| 密码最近修改时间  | 基本信息区 | 否     | 仅展示            |
| 原密码       | 安全设置区 | 是     | 修改密码时填写        |
| 新密码       | 安全设置区 | 是     | 修改密码时填写        |
| 确认新密码     | 安全设置区 | 是     | 修改密码时填写        |
| 当前登录时间    | 会话概览区 | 否     | 仅展示            |
| 最近活跃时间    | 会话概览区 | 否     | 仅展示            |
| 当前会话状态    | 会话概览区 | 否     | 仅展示，如在线/即将超时   |
| 会话剩余有效时长  | 会话概览区 | 否     | 仅展示            |

### 10.5 页面交互说明

#### 10.5.1 基本信息区

- 以只读表单或信息卡片形式展示。
- 用户不可在本页面修改用户名、角色、工段、账号状态。
- 当用户为非操作员角色时，工段展示为 `/`。

#### 10.5.2 安全设置区

- 原密码、新密码、确认新密码均为密码输入框。
- 新密码输入时，页面应展示密码规则提示。
- 提交保存前应进行前端校验：
  - 原密码不能为空
  - 新密码不能为空
  - 确认新密码不能为空
  - 新密码与确认新密码必须一致
  - 新密码不能与原密码相同
- 提交成功后弹出成功提示，并强制当前用户重新登录。
- 提交失败时需展示明确失败原因，如原密码错误、新密码不符合规则、两次输入的新密码不一致等。

#### 10.5.3 会话概览区

- 展示当前登录会话摘要信息。
- 若会话接近超时时间，应进行提示。
- 支持“退出当前登录”按钮，点击后当前登录态立即失效并返回登录页。

### 10.6 业务规则

- 所有已登录用户均可访问个人中心/账号设置页面。
- 用户仅可查看和维护本人信息，不可查看其他用户个人中心内容。
- 用户不可在个人中心修改自己的用户名、角色、工段、账号状态、权限信息。
- 修改密码时必须校验原密码正确性。
- 新密码必须符合系统密码规则。
- 新密码不得与原密码相同。
- 新密码与确认新密码必须一致。
- 修改密码成功后，当前登录态立即失效，用户必须重新登录。
- 若会话已过期，页面操作应提示登录失效并跳转登录页。

### 10.7 安全规则

- 密码输入框默认掩码显示。
- 修改密码操作需写入操作审计日志。
- 页面不得缓存原密码、新密码、确认新密码明文。

## 11. 登录日志/在线会话页面

### 11.1 页面目标

用于查看登录记录和当前在线会话。

### 11.2 页面组成

包含两个页签：

1. 登录日志
2. 在线会话

### 11.3 登录日志

用于查看账号登录历史记录。

#### 11.3.1 列表字段

| 表头   | 说明      |
| ---- | ------- |
| 用户名  | 登录账号    |
| 登录时间 | 登录发生时间  |
| 登录结果 | 成功/失败   |
| 失败原因 | 登录失败时展示 |

#### 11.3.2 查询条件

- 用户名
- 登录结果
- 时间范围

#### 11.3.3 业务规则

- 登录日志记录登录成功与失败记录。
- 登录日志保存 30 天。
- 超过 30 天的登录日志执行物理删除。

### 11.4 在线会话

用于查看当前在线用户会话状态。

#### 11.4.1 列表字段

| 表头     | 说明       |
| ------ | -------- |
| 选择框    | 用于批量强制下线 |
| 用户名    | 当前在线用户   |
| 角色     | 当前用户角色   |
| 工段     | 当前用户工段   |
| 登录时间   | 本次登录时间   |
| 最近活跃时间 | 最近一次操作时间 |
| 状态     | 在线/离线    |
| 操作     | 强制下线     |

#### 11.4.2 操作说明

- 支持对单个在线会话执行“强制下线”。
- 支持批量强制下线。
- 强制下线后，被下线用户当前登录状态立即失效，需重新登录。

#### 11.4.3 业务规则

- 在线会话最长有效时长为 1 小时。
- 在线会话状态应与系统登录状态保持一致。
- 强制下线操作需记录审计日志。
- 强制下线权限仅系统管理员拥有。
- 页面查看权限由功能权限配置决定；若无页面权限则不可进入本页面。
- 普通用户不可查看全部在线会话。

## 12. 通用业务规则

### 12.1 用户名规则

- 用户名全局唯一。
- 用户名长度为 2～10 位。
- 用户名允许中文。
- 用户名不区分大小写。

### 12.2 密码规则

- 密码长度不少于 6 位。
- 密码可以包含数字、字母、符号。
- 支持管理员重置密码。
- 支持个人中心修改密码。

### 12.3 在线状态规则

- 用户登录成功且会话有效时，状态为“在线”。
- 用户主动退出登录后，状态为“离线”。
- 会话超过 1 小时后自动失效。
- 在线显示绿色；离线显示灰色。

### 12.4 角色与工段联动规则

- 操作员角色必须绑定工段。
- 非操作员角色不可绑定工段。
- 角色从“操作员”修改为其他角色时，工段自动清空。
- 角色修改为“操作员”时，需重新选择工段。
- 工段数据由工艺模块维护，用户模块仅负责引用。
- 工段下拉数据实时查询工艺模块。

### 12.5 删除规则

- 用户删除统一采用逻辑删除。
- 停用与删除均保留历史数据。
- 删除操作必须二次确认，并记录日志。

### 12.6 权限安全规则

- 至少保留一个具备功能权限配置页面访问能力的系统管理员账号。
- 不允许通过错误操作导致系统中无人可进入功能权限配置页面。
- 权限修改必须记录审计日志。
- 本期不包含数据范围权限。
