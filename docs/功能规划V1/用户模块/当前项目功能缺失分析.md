# 当前项目功能规划

## 用户模块

- ### 1、用户管理页面
  
  - 搜索功能（用于搜索全用户展示列表，搜索目标为用户名）
  
  - 新建角色功能
    
    - 系统自带角色
      
      - 系统管理员
      
      - 生产管理员
      
      - 品质管理员
      
      - 操作员
      
      - 维修员
  
  - 新建用户功能
    
    - 用户名
    
    - 用户密码
    
    - 用户角色（单选）
    
    - 用户工段（单选、仅操作员角色可选）
  
  - 全用户展示列表展示功能（有表头、可滚动，按照用户在数据库中的 id 字段排序）
    
    - 需有以下表头
      
      - 用户名
      
      - 角色
      
      - 工段（未分配、无法分配角色留空或斜杠）
      
      - 在线状态
        
        - 在线（绿色）
        
        - 离线（灰色）
      
      - 操作
        
        - 操作按钮
          
          - 下拉菜单中包含
            
            - 编辑
            
            - 删除

- ### 2、注册审批页面
  
  - 全注册审批用户展示功能（有表头、可滚动，按照用户在数据库中的 id 字段排序）
    
    * 需有以下表头
      
      * 用户名
      
      * 密码
      
      * 操作
        
        * 操作按钮
          
          * 下拉菜单中包含
            
            * 通过
              
              - 用户名
              
              - 用户密码
              
              - 用户角色（单选）
              
              - 用户工段（单选、仅操作员角色可选）
            
            * 驳回
              
              - 确认弹窗

- 3、功能权限配置页面（去掉搜索、当前版本显示、生效预览、查看变更历史、查看最近保存结果、恢复上次保存前草稿、套用岗位模板功能）
  
  - 模块选择下拉栏
    
    - 用户模块
    
    - 产品模块
    
    - 工艺模块
    
    - 生产模块
    
    - 品质模块
    
    - 设备模块
    
    - 消息模块
  
  - 角色选择下拉栏
  
  - 能力包展示功能
  
  ```
  注意：
      1、系统管理员不再固定全权限，只固定拥有功能权限配置页面权限，功能权限配置页面权限仅系统管理员角色可拥有、编辑，不出现在能力包中。
  ```
