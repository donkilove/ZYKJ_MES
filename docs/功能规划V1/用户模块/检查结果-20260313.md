### 1.2 通用业务规则实现情况

| 规则 | 状态 | 说明 |
|------|------|------|
| 用户名全局唯一 | ✅ | 数据库唯一约束 + 应用层校验 |
| 用户名允许中文 | ✅ | Unicode 字符串支持 |
| 用户名不区分大小写 | ✅ | `func.lower()` 比较 |
| 用户名长度 2-10 位 | ✅ | Schema 定义 `min_length=2, max_length=10` |
| 密码不少于 6 位 | ✅ | `min_length=6` |
| 密码不得连续 4 位相同字符 | ✅ | `validate_password()` 函数实现 |
| 密码不支持与已有用户相同 | ✅ | `validate_password()` 遍历比对哈希 |
| 首次登录强制修改密码 | ✅ | `must_change_password` 字段 |
| 在线会话最大时长 1 小时 | ✅ | `session_max_seconds=3600` |
| 登录日志保留 30 天 | ✅ | `login_log_retention_days=30` |
| 操作员必须绑定工段 | ✅ | `_resolve_stage()` 校验 |
| 非操作员不可绑定工段 | ✅ | `_resolve_stage()` 校验 |
