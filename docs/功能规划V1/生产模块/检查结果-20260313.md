## 4. 剩余 P1 问题（后续迭代）

1. 代班记录筛选条件不全（仅状态，缺订单号/工序/发起人/代班人/时间范围）
2. 代班记录列表字段不全（缺审批人、审批时间、查看详情）
3. 报废处理进度口径与需求不一致（实现为"待申请/已申请"，需求为"待处理/已处理"）
4. 维修详情当前为弹窗，不是独立子页面；未体现"相关事件记录"区
5. 生产数据"今日实时"未明确展示"最后刷新时间"字段
