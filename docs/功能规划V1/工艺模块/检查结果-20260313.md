# 工艺模块需求实现检查结果

- 检查日期：2026-03-14
- 对照文档：`docs/功能规划V1/工艺模块/工艺模块需求说明.md`
- 检查范围：后端（FastAPI）+ 前端（Flutter）+ 跨模块联动 + 需求文档
- 检查方式：静态代码核查，结合接口、服务、页面、数据模型与跨模块调用逐项比对；本次未做人工页面点测或联调录屏
- 检查人：`OpenCode`

---

## 2. 检查结论

当前工艺模块对需求说明的满足情况：`部分满足`

结论摘要：

- 后端主体能力已具备：工段/工序 CRUD、系统母版、模板创建编辑、模板发布、版本记录、版本对比、回滚、JSON 导入导出、工艺看板接口都能在代码中确认。
- 关键 P0 差距仍然存在：独立“工艺引用分析页面”未落地，工段/工序引用分析接口缺失，工段删除保护未覆盖用户引用，页面侧也缺少“查看引用”入口。
- 工段/工序管理页、模板配置页、工艺看板页都不是空白页，但与需求文档仍有多处结构性差异，当前不宜按“完全满足需求说明”结论验收。

## 6. 未满足或与需求不一致的关键项

### 6.1 工段/工序管理页面

- 页面未提供需求要求的搜索/筛选与导出能力；当前只加载全量工段/工序，顶部只有“新增工段”“新增工序”“刷新”。  
  证据：`mes_client/lib/pages/process_management_page.dart:65`、`mes_client/lib/pages/process_management_page.dart:71`、`mes_client/lib/pages/process_management_page.dart:75`、`mes_client/lib/pages/process_management_page.dart:741`
- 工段列表与工序列表表头字段不完整，缺少“关联工序数”“创建时间”“查看引用”等文档要求内容。  
  证据：`mes_client/lib/pages/process_management_page.dart:691`、`mes_client/lib/pages/process_management_page.dart:707`、`mes_client/lib/pages/process_management_page.dart:855`、`mes_client/lib/pages/process_management_page.dart:964`
- 工段名称唯一、同一工段下工序名称唯一未在后端落实；当前仅校验工段编码唯一与工序编码唯一。  
  证据：`backend/app/services/craft_service.py:203`、`backend/app/services/craft_service.py:227`、`backend/app/services/craft_service.py:298`、`backend/app/services/craft_service.py:322`
- 删除保护不完整：工段删除未校验用户模块对工段的绑定引用，工序删除未校验历史工艺版本或系统母版引用，也没有删除前引用分析展示。  
  证据：`backend/app/services/craft_service.py:251`、`backend/app/services/craft_service.py:345`、`backend/app/models/user.py:27`

### 6.2 工艺模板配置页面

- 模板复制/套版未落地：前端模板操作只有编辑、发布、影响分析、版本管理、删除，没有复制/套版；后端也未见对应 API。  
  证据：`mes_client/lib/pages/process_configuration_page.dart:22`、`mes_client/lib/pages/process_configuration_page.dart:2071`、`backend/app/api/v1/endpoints/craft.py:486`
- 模板归档/取消归档未形成动作闭环：生命周期值支持 `archived`，前端可筛选“已归档”，但未见 archive/unarchive 接口或操作入口。  
  证据：`backend/app/schemas/craft.py:5`、`mes_client/lib/pages/process_configuration_page.dart:1964`、`mes_client/lib/pages/process_configuration_page.dart:2071`
- 系统母版维护只有当前版本，未实现“查看历史版本”与“复制为产品模板”显式能力。  
  证据：`backend/app/api/v1/endpoints/craft.py:395`、`backend/app/api/v1/endpoints/craft.py:406`、`backend/app/api/v1/endpoints/craft.py:423`、`backend/app/models/craft_system_master_template.py:7`
- 模板列表筛选条件明显少于需求说明，当前页面只有“按产品筛选”和“按生命周期筛选”。  
  证据：`mes_client/lib/pages/process_configuration_page.dart:49`、`mes_client/lib/pages/process_configuration_page.dart:1926`

  证据：`backend/app/services/craft_service.py:1461`、`backend/app/schemas/craft.py:347`
- 发布请求未携带版本戳或并发校验字段，未体现“他人已修改则提示刷新重试”的机制。  
  证据：`backend/app/schemas/craft.py:232`、`backend/app/services/craft_service.py:1105`
