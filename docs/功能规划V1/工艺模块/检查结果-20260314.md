## 各页面详细检查

### 1. 工段/工序管理（P0）— ✅ 基本完成

**后端实现：**
- `craft.py`：工段列表/新增/编辑/删除（14 个端点全部实现）、工序列表/新增/编辑/删除、工段导出、工序导出、工段引用查询、工序引用查询

**前端实现：**
- `process_management_page.dart`：工段列表（含启用/禁用筛选、导出、引用分析弹窗）、工序列表（含工段筛选、导出）、新增/编辑/删除对话框

**缺失功能点：**
- 工段/工序模型缺少"备注(remark)"字段
- 工段 `name` 字段缺少唯一约束（仅 `code` 有唯一约束）
- 删除前未先展示引用分析结果（仅简单确认框）

### 2. 模板配置（P0）— ✅ 基本完成

**后端实现：**
- `craft.py`：模板列表、新增、编辑、删除、详情、发布、版本列表、版本对比、回滚、复制、跨产品复制、从母版复制、归档/取消归档、系统母版 CRUD、模板导出/导入、影响分析（19 个端点全部实现）

**前端实现：**
- `process_configuration_page.dart`：系统母版区、产品模板列表、新增/编辑/删除、发布（含影响分析）、版本记录、复制、归档、导出/导入

**缺失功能点：**
- 模板操作菜单缺少"回滚"入口（后端 API 和前端 service 已实现，仅菜单未接入）
- 模板缺少"备注(remark)"字段
- 缺少多项筛选条件：模板名称搜索、产品分类筛选、默认模板筛选、状态筛选、更新时间范围筛选
- 列表缺少"最近更新人"列
- 系统母版历史版本查看功能未实现
- 模板步骤缺少标准工时、关键工序标识、步骤说明字段（需求标注 P1 可延后）
- 并发发布冲突检测（乐观锁）未实现

### 3. 引用分析（P1）— ✅ 基本完成

**后端实现：**
- `craft.py`：工段引用查询、工序引用查询、模板引用查询、模板影响分析（4 个端点全部实现）

**前端实现：**
- `craft_reference_analysis_page.dart`：三维度查询（工段/工序/模板）、左侧选择面板（含搜索）、右侧引用结果（引用类型标签、风险等级标签）

**缺失功能点：**
- 缺少"按产品查询模板引用"维度
- 缺少"引用状态"列（正在使用/历史引用/可同步/不可同步）
- 缺少"跳转操作"（查看详情或跳转来源模块）
- 未查询设备/统计/规则配置中的引用

### 4. 看板（P1）— ✅ 基本完成

**后端实现：**
- `craft.py`：工艺看板工序指标（`/kanban/process-metrics`）

**前端实现：**
- `craft_kanban_page.dart`：产品选择、时间范围筛选、工段/工序联动筛选、工时柱状图、产能折线图

**缺失功能点：**
- 看板数据导出功能未实现
