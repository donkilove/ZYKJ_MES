# 前端测试用例清单

## 1. 模型层用例

### 1.1 会话与用户
- `models/app_session_test.dart`
  - 校验 `AppSession` 字段赋值。
- `models/current_user_test.dart`
  - 校验 `fromJson` 映射。
  - 校验 `displayName` 优先显示姓名、空姓名回退账号。
- `models/user_models_test.dart`
  - 校验用户、角色、工序、注册请求模型解析。
  - 校验列表包装结构字段。

### 1.2 产品与质量
- `models/product_models_test.dart`
  - 校验产品、参数、历史、更新结果模型解析与序列化。
- `models/quality_models_test.dart`
  - 校验首件检验结果/验证码来源文案函数分支。
  - 校验时间字段容错默认值。

### 1.3 工艺、生产、设备、页面目录
- `models/craft_models_test.dart`
  - 校验工段/工序/模板/系统母版模型解析。
  - 校验模板同步结果模型解析。
- `models/production_models_test.dart`
  - 校验订单、明细、统计、操作结果、步骤输入序列化。
  - 校验状态文案函数分支。
- `models/equipment_models_test.dart`
  - 校验设备、保养项、计划、工单、记录模型解析。
  - 校验周期常量与执行日期文案分支。
- `models/page_catalog_models_test.dart`
  - 校验页面目录模型解析。
  - 校验 fallback 目录常量有效性。

## 2. 服务层用例

- `services/api_exception_test.dart`
  - 校验异常对象字段与 `toString`。
- `services/auth_service_test.dart`
  - 登录/注册/账户列表/当前用户/登出成功链路。
  - 登录失败、缺失 token 异常分支。
- `services/user_service_test.dart`
  - 用户列表、角色、工序、注册审批、用户增改删全链路。
  - 非 200 响应异常分支。
- `services/product_service_test.dart`
  - 产品列表、创建、删除、参数查询、参数更新、参数历史。
  - 创建失败异常分支。
- `services/quality_service_test.dart`
  - 首件列表、总览/工序/人员统计查询。
  - 日期格式与分页上限处理。
  - 失败分支异常处理。
- `services/craft_service_test.dart`
  - 工段/工序 CRUD、模板查询与编辑、系统母版 CRUD、模板同步结果。
  - 母版为空分支。
  - 非 JSON 错误响应分支。
- `services/equipment_service_test.dart`
  - 设备台账、保养项、计划、执行、记录全链路。
  - 禁用快捷方法（`disable*`）分支。
  - 非 200 响应异常分支。
- `services/production_service_test.dart`
  - 生产订单 CRUD、完工、首检、报工、详情、我的工单、统计、选项查询。
  - 日期格式化与分页上限分支。
  - 非 200 响应异常分支。
- `services/page_catalog_service_test.dart`
  - 页面目录查询与异常分支。
  - 非 200 响应异常分支。
- `services/session_store_test.dart`
  - 会话保存/读取/清理与空 token 分支。

## 3. 组件层用例

- `widgets/locked_form_dialog_test.dart`
  - `Esc` 不关闭弹窗。
  - 通过显式按钮关闭弹窗。
- `widgets/adaptive_table_container_test.dart`
  - 容器渲染、双滚动区域结构校验。

## 4. 支撑工具
- `support/http_test_server.dart`
  - 本地 HTTP 模拟服务，用于服务层契约测试与请求断言。
