# 05_产品参数管理与查询业务逻辑

## 1. 模块职责与边界
- 职责：维护产品参数、参数可见性、参数修改历史与备注，支持查询与导出。
- 边界：产品主档维护不在本模块（见 04）。

## 2. 入口与调用链
- 管理页：`src/ui/son_page/product_parameter_management_page.py`
- 查询页：`src/ui/son_page/product_parameter_query_page.py`
- 相关弹窗：
  - `add_parameter_window.py`
  - `edit_parameter_window.py`
  - `edit_product_parameters_window.py`
  - `view_product_parameters_window.py`
  - `export_parameters_window.py`

调用链：
`UI -> parameter_controller/product_controller -> product_service -> product_impl`

## 3. 业务对象
- 参数主表：`product_parameter_list`
  - `param_name` `param_category` `param_type` `param_value` `process_name` `product_id`
- 可见性表：`parameter_visibility`
- 历史审计表：`product_parameter_history`、`parameter_remarks`

## 4. 核心流程
### 4.1 参数新增
1. 选择产品。
2. 输入参数名称、分类、类型、值、工序、备注。
3. 保存参数主记录。
4. 记录备注与历史。
5. 保存角色可见性。

### 4.2 参数编辑
1. 加载参数详情。
2. 更新名称/分类/类型/工序/值。
3. 记录历史（旧值、新值、备注、时间）。
4. 更新可见角色。

### 4.3 参数查询
- 按产品加载参数。
- 按角色过滤可见项。
- Link 类型可调用系统打开地址。

### 4.4 参数导出
- 支持导出到指定路径（Excel/文本，取决于实现入口）。
- 导出内容遵循当前筛选结果。

## 5. 关键函数索引（核心）
- `src/controller/parameter_controller.py`
  - `load_parameters` `add_parameter` `update_parameter` `delete_parameter`
- `src/service/product_service.py`
  - `add_parameter_with_visibility` `update_parameter` `save_parameter_modification` `save_parameter_visibility`
- `src/impl/product_impl.py`
  - `add_parameter_with_visibility` `update_parameter` `save_parameter_modification` `get_product_parameter_history` `open_parameter_link`

## 6. 数据库交互
- 主写入：`product_parameter_list`
- 审计写入：`product_parameter_history`、`parameter_remarks`
- 角色可见：`parameter_visibility`
- 读取时按 `product_id` + 角色组合过滤

## 7. 权限与可见性
- 参数查询页对所有角色开放（系统硬规则）。
- 参数管理页由角色可见性配置决定。
- 参数可见性支持细粒度到“角色-参数”级别。

## 8. 并发与一致性
- 参数修改后清理缓存并刷新页面。
- 历史记录与参数主记录建议同事务提交。
- Link 打开为客户端行为，不改变数据库状态。

## 9. 失败路径
- 参数字段缺失：保存失败。
- 参数不存在：编辑/删除失败。
- 导出路径非法：导出失败。
- Link 地址不可达：打开失败提示。

## 10. 模块耦合
- 与产品主档：参数依赖产品ID。
- 与订单：订单查询页按工序读取参数。
- 与权限：可见性受角色矩阵+参数可见表双重控制。

## 11. 回归测试（10条）
1. 新增参数后管理页可见。
2. 查询页能看到当前角色可见参数。
3. 修改参数后历史增加。
4. 删除参数成功。
5. 参数备注可正确保存。
6. 角色可见性限制生效。
7. Link 参数点击可打开目标。
8. 导出参数成功生成文件。
9. 参数搜索结果正确。
10. 多次编辑后历史时间线连续。
