# 项目记忆入口

本目录用于保存适合进入 Graphify 项目结构记忆、且有长期复用价值的稳定项目事实。

## 1. 记录范围

- 稳定架构决策、模块边界、跨模块依赖关系。
- 长期有效的运行约定、工具约定、环境约定。
- 已由源码、测试、接口契约或 evidence 验证过的业务流程事实。
- 对后续排查、设计、迁移、联调有复用价值的项目级结论。

## 2. 禁止记录

- 密钥、令牌、密码、内部敏感链接、个人隐私数据。
- 一次性调试输出、临时日志、未验证猜测。
- 可直接从 `evidence/`、`progress.md` 读取的任务流水细节。
- 已被 `.graphifyignore` 排除的构建产物、缓存和导出文件。

## 3. 更新口径

- 不要求每次任务都写入本目录；只有结论会被后续排查、设计、迁移或联调反复使用时才写。
- 写入后按需在 `evidence/`、`progress.md` 或最终回复中记录 Graphify 回写动作、是否刷新图谱，以及未刷新时的原因。
- 若只改代码文件且本任务依赖图谱，可运行 `graphify update .` 做 AST-only 刷新。
- 若新增或修改本目录 Markdown，可记录“待语义重抽取”或按明确范围执行 Graphify 语义更新。

## 4. 事实源边界

本目录是 Graphify 的可摄取项目记忆入口，不是最终事实源。若与源码、测试、接口契约、`AGENTS.md` 或 `evidence/` 冲突，以后者和真实验证为准，并更新本目录。
