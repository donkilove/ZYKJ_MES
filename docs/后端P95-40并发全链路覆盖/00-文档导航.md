# 后端 P95-40 并发全链路覆盖文档导航

## 1. 本目录解决什么问题

本目录用于统一回答以下问题：

- 后端 P95-40 并发当前到底覆盖到了哪里。
- “命中全覆盖”和“有效全覆盖”有什么区别。
- 当前哪些结果可以用于性能验收，哪些只能用于覆盖审计。
- 后续如果要持续回归，应该怎么执行、怎么看结果、优先补什么。

## 2. 当前一句话结论

- 截至 2026-04-11，`40` 并发全链路场景对后端已实现**命中全覆盖**。
- 但从“成功返回并可用于性能验收”的口径看，**尚未实现有效全覆盖**。
- 目前可稳定用于性能门禁的，是读链路专项 `91` 场景集合；该集合最近一次结果为 `P95=438.11ms`、`error_rate=1.01%`、`gate_passed=true`。
